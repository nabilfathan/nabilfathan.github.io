📚 Informasi Media Populer

🔠 Font Populer

  • Times New Roman: Font serif klasik untuk dokumen resmi dan akademik.
  • Arial: Font sans-serif sederhana dan bersih.
  • Helvetica: Populer di desain grafis dan branding.
  • Calibri: Font default Microsoft Office, sangat terbaca.
  • Roboto: Font sans-serif modern yang digunakan di Android.

🖼️ Format Gambar Populer

  • JPEG (.jpg/.jpeg): Format lossy untuk foto, ukuran kecil.
  • PNG (.png): Mendukung transparansi, cocok untuk ikon.
  • GIF (.gif): Animasi ringan, banyak digunakan di web.
  • SVG (.svg): Format vektor yang dapat diskalakan.
  • WEBP (.webp): Kompresi modern dengan kualitas bagus.

🎞️ Format Video Populer

  • MP4 (.mp4): Format serbaguna dan paling umum digunakan.
  • AVI (.avi): Format lama dengan sedikit kompresi.
  • MKV (.mkv): Mendukung banyak codec dan subtitle.
  • MOV (.mov): Kualitas tinggi, cocok untuk profesional.
  • WEBM (.webm): Format ringan untuk video web.

🔊 Format Audio Populer

  • MP3 (.mp3): Format audio umum dengan ukuran kecil.
  • WAV (.wav): Audio berkualitas tinggi, tanpa kompresi.
  • FLAC (.flac): Kompresi lossless dengan kualitas tetap.
  • AAC (.aac): Lebih efisien dari MP3, digunakan di iTunes.
  • OGG (.ogg): Format open-source untuk audio.

🎬 Jenis Animasi Populer

  • 2D Animation: Animasi datar, seperti kartun klasik.
  • 3D Animation: Menggunakan objek 3D, umum di film dan game.
  • Stop Motion: Animasi dari foto objek nyata secara berurutan.
  • Motion Graphics: Animasi teks dan grafis untuk presentasi/video.
  • Whiteboard Animation: Gaya gambar di papan putih, sering untuk edukasi.
